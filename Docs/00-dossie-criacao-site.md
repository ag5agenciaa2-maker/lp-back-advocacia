*Justificativa:* a Sandra tem dois públicos distintos (empresas + famílias com direitos de saúde/previdência). Cada bloco vira uma página de livro: empresa à esquerda + imagem mármore preto à direita; família à direita + imagem mais quente à esquerda.

### DEPOIMENTOS — escolhido
**[X] E) Carrossel fade + nota Google visível**

*Justificativa:* nicho jurídico exige prova social autenticada (não inventada). Se não atender, bloco é substituído por seção "Princípio" editorial.

### SOBRE / CREDENCIAIS — escolhido

### ANIMAÇÕES DO PROJETO

- **Topbar** → ao rolar mais de 20px: box-shadow `0 4px 20px -10px rgba(10,22,40,.18)` aparece em 350ms, easing `cubic-bezier(.2,.7,.2,1)`, trigger: scroll
- **Hero h1** → de `opacity:0; translateY(20px)` para `opacity:1; translateY(0)` em 900ms, easing `cubic-bezier(.2,.7,.2,1)`, trigger: load, stagger: não
- **Hero lede + CTA + selos** → mesma animação do h1, stagger: sim, delay incremental de 150ms entre cada elemento
- **Hero imagem (clip-path)** → de `clip-path: polygon(20% 0, 100% 0, 100% 100%, 0 100%)` para `clip-path: polygon(10% 0, 100% 0, 100% 100%, 0 100%)` em 1200ms, easing `cubic-bezier(.2,.7,.2,1)`, trigger: load
- **Cards de serviços (zig-zag)** → de `opacity:0; translateY(40px)` para `opacity:1; translateY(0)` em 800ms, easing `cubic-bezier(.2,.7,.2,1)`, trigger: IntersectionObserver threshold 0.15
- **Carrossel depoimentos** → cada slide: de `opacity:0` para `opacity:1` em 600ms, easing `ease-in-out`, trigger: timer 6s, stagger: não. Pausa automática ao hover.
- **Foto Sandra (sticky)** → no scroll do bloco, `transform: scale(1) → scale(1.02)` em 1500ms linear, trigger: scroll do container pai
- **FAQ accordeon** → summary marker `+` rotaciona 45° para virar `×` em 350ms easing `ease`, trigger: open
- **WhatsApp FAB** → de `opacity:0; scale(.6)` para `opacity:1; scale(1)` em 400ms easing cubic-bezier(.2,.9,.3,1.4), trigger: scroll > 80px ou timer 8s
- **Balão WhatsApp** → de `opacity:0; translateY(20px) scale(.96)` para `opacity:1; translateY(0) scale(1)` em 500ms easing cubic-bezier(.2,.9,.3,1.2), trigger: timer 15s após FAB
- **Typing dots no balão** → 3 bolinhas com animação `translateY(0) ↔ translateY(-4px)` em 1200ms ease-in-out infinite, delay incremental 0.15s
- **Mensagem do balão** → fade `opacity:0 → opacity:1` em 500ms ease, trigger: 2500ms após abertura do balão
- **Badge "1" pós-close** → de `scale(0)` para `scale(1)` em 350ms easing cubic-bezier(.2,.9,.3,1.6), trigger: 2000ms após fechar o balão
- **Reveal genérico de seções** → `.section__title`, `.section__lede`, `.card`, `.manifesto__text`, `.sandra__inner`, `.faq__item`, `.contato__grid`: de `opacity:0; translateY(24px)` para `opacity:1; translateY(0)` em 800ms easing cubic-bezier(.2,.7,.2,1), trigger: IntersectionObserver threshold 0.12, rootMargin `0px 0px -60px 0px`
